Project Documentation

LBYCPG3 Final Project

About the Developers

We are a team of 5th-year Computer Engineering students from De La Salle University Manila, graduating in 2026. This project was developed as a final requirement to demonstrate modern front-end web development principles.

Project Relevance

This website addresses the UN Sustainable Development Goal 13 (Climate Action) with a strict local focus on the Philippines. By providing an interactive tool for disaster preparedness, we aim to make typhoon readiness more accessible and actionable for households in Metro Manila and beyond.

Technical Stack

  • Structure: Semantic HTML5 (5 separate pages)
  • Styling: Bootstrap 5 Grid system and Custom CSS transitions
  • Interactivity: Vanilla JavaScript for dynamic DOM manipulation (Progress Bar calculations)
  • Backend: Supabase integration for cloud data persistence of user kit scores
  • Design: Initial mockups created using Figma
